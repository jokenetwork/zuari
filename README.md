zuari
===

A WordPress theme to log your stream of life. To get started:

```
npm install
grunt
```

## Stuff to delete before uploading
Before you upload the theme to [WordPress.org](https://wordpress.org/themes/upload/), delete the following:

```
.git
.gitignore
.sass-cache
Gruntfile.js
README.md
style.css.map
node_modules
**/*.scss
package.json
package-lock.json
composer.json
composer.lock
vendor
.stylelintrc.json
```
